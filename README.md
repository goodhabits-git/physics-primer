# Physics Primer

This is an interactive, beginner-friendly physics primer focused on building intuition for basic mechanics. The goal of this project is not to replace lectures or textbooks, but to help students understand core ideas in a more visual and interactive way before or alongside other study. This is for students and hobbyists alike, but a certain level of mathematical proficiency is expected before engaging with it. You should be comfortable with basic algebra and trigonometry. Familiarity with calculus will help you a lot, but it is gone over in the primer as necessary.


## Interactive Elements

This project includes small JavaScript-based simulations that help demonstrate concepts in a hands-on way. It also has various practice problems with solutions to get you started problem solving.
